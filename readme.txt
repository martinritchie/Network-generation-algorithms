README: Network-generation algorithms

This repository contains the Matlab code and files for the two proposed network generation algorithms as described in: http://arxiv.org/abs/1512.01435

Included files: CMA.m, UDA.m, Connect_repeated.m, Connect_refused.m, Connect_erased.m, clustering.m

Please refer to the quick start files, that are matlab scripts, which provide example of thier use.

Written by Martin Ritchie, University of Sussex, 2016.
